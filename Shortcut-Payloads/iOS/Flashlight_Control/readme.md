## Flash light controller 

Simple shortcut to use the iOS C2 to control the flash light.

## Requirements: 

None, no permission promptps. 

## Usage: 

<div align="center">

  |     Command     |     Useage     |
|:-----------------:|:-----------------:|
| flashlight:  | Ingores, doesnt turns flashlight off.  |
| device_details:1  | Turns the flashlight on. |


</div>
